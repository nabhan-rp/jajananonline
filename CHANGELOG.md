
# Changelog Jajanan Online

Semua perubahan pada proyek **Jajanan Online** (Rebrand dari QiosLink) didokumentasikan di sini.

**Current Version:** `v1.0`
**Release Date:** 14 Januari 2026

---

## [v1.0] - 2026-01-14
### Initial Release
- **Rebranding:** Perubahan total UI/UX menjadi "Jajanan Online" dengan tema Orange.
- **New Product Types:** 
  - **License Keys:** Dukungan untuk penjualan serial key unik (stok otomatis berkurang).
  - **Digital Static:** Dukungan untuk Ebook/File link.
  - **Physical:** Dukungan pengiriman barang fisik.
- **Review System:** Fitur rating bintang 5 dan komentar untuk user yang sudah membeli (Verified Purchase).
- **Embedded QRIS:** Integrasi engine pembayaran QiosLink secara native di dalam sistem order.
- **Auto Delivery:** Serial key dikirim otomatis setelah status pembayaran 'paid' via callback.
